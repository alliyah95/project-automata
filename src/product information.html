Project Automata

FEATURES &

BENEFITS


Automata is a program that runs automatically after a few clicks once inserted
in the USB port of a computer. It does not need internet connection to work. Also, no additional files are
required to make the product performs its functions.


CLEANUP

Once executed by the program, it will
mainly run a third-party application
that comes pre-installed with the flash
drive. Moreover, it will also execute
system commands that would target
unnecessary files such as junk and
temporary files.

PROTECTION

Responsible for running a low-key anti-
malware that comes pre-installed with the
flash drive.This would be a basic-level
protection that would only target common
malware that users experience frequently.

RESTORATION

The function that takes the longest to
execute and is mainly optional. It is a script of
system commands built-in withinthe Windows
10 Operating System and restores the
system to its former, much better and faster
state as much as possible or as much as the
system could allow.

No additional
files needed.

Automata has pre-installed third party applications
which saves the user a lot of time as well as protects
them from potential malware attacks.

Faster and Easier
Computer Maintenance

Automata allows the user to resume their work while having
their system optimized and brought back to its former glory.

Portability
and Simplicity

No user interface. Just a small storage device to
power up your computer.
